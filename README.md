# AI Japanese Chatbot Demo

A simple AI-powered Japanese conversation practice tool.

## Features
- Scenario-based conversation
- AI-generated feedback
- Beginner-friendly interaction

## How to run

1. Install dependencies
pip install flask requests

2. Set API key
export OPENAI_API_KEY=your_key

3. Run
python app.py

4. Open browser
http://127.0.0.1:5000
